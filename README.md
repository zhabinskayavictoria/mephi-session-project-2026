# Сессионный проект по курсу "Операционные системы семейства Unix"

## Краткое описание

В рамках задания необходимо подготовить минимальную установку Fedora 43 к работе в локальной сети МИФИ, настроить безопасность в соответствии с политиками и развернуть простой web-сервер для публикации модели (в будущем). Все настройки должны сохраняться после перезагрузки/восстановления системы.

## Состав репозитория

| Файл | Описание |
|------|----------|
| `project_history.txt` | История всех команд |
| `network_check.txt` | Результаты ping до 8.8.8.8 и 192.168.1.1 |
| `nginx_recent_logs.txt` | Логи nginx |
| `fstab.txt` | Содержимое /etc/fstab |
| `selinux_status.txt` | Режим SELinux |
| `file_contexts.txt` | Контекст SELinux |
| `tcpdump_capabilities.txt` | Настройка capabilities |
| `permissions.txt` | Права и владелец директории /data/mephi-web |
| `users_groups.txt` | Пользователи и группы |
| `index.html` | Веб-страница |
| `curl_output.txt` | Вывод curl http://192.168.1.100 |
| `mephi-nginx-screenshot.png` | Скриншот с результатом работы |
| `tcpdump.rpm` | Скачанный RPM-пакет tcpdump |
